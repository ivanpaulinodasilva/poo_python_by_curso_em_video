# Curso de Python Orientado a Objetos (POO) 🐍

Repositório dedicado aos estudos do curso de Programação Orientada a Objetos com Python, ministrado pelo Professor Gustavo Guanabara no canal **Curso em Vídeo**.

## 📖 Sobre o Módulo
Este módulo é considerado o "Mundo 4" do curso de Python. O foco é sair da programação estruturada e entender o paradigma da Orientação a Objetos de forma pura e simples.

### 🏗️ Estrutura do Conteúdo
1. **Embasamento:** Evolução das tecnologias e por que a POO é necessária.
2. **Fundamentos:** Classes, Objetos, Atributos, Métodos, Estado e Instanciação.
3. **Os 4 Pilares:**
   - Abstração
   - Encapsulamento
   - Herança
   - Polimorfismo
4. **Conceitos Avançados:** Getters/Setters, `self`, `super`, Métodos Mágicos (Dunder Methods) e Relacionamentos (Associação, Agregação e Composição).
5. **Projeto Final:** Acesso a banco de dados (SQLite3) e introdução a ORM (Object Relational Mapper).
